# Online Queue Management System – Business Analysis Project

## Project Overview

This project analyzes and designs an **Online Queue Management System for outpatient hospital services**.

The goal of the system is to reduce waiting time for patients and improve hospital workflow by allowing patients to **take queue numbers remotely and schedule medical appointments through a mobile application**.

The system integrates with the hospital’s existing patient management system to synchronize medical records and appointment data.
